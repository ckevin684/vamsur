## ⚡ 가장 시급한 문제들 (P0 — 즉시 조정 필요)

### 🚨 P0-1: 황기훈 SAED 보스 한 방 컷
- **현황**:
  - Lv8 caff 도끼 모드 hit 데미지: ~530 (베이스 dmg × 4.5 + 번개 3발 × 1.8x)
  - cd 0.7s × CD_M[7] = 0.455s (Lv8)
  - DPS: ~1158 단일 타깃
  - 보스 보정: gh 베이스 bdm 1.25 + monhun 만렙 ×2.49 = **bdm 3.11**
  - vs lol 보스 HP 8000: **DPS 3601 → TTK ~2.2초** (페이즈 전환도 무시)
- **권장**: 
  - 번개 데미지 ×1.8 → ×1.2 또는
  - 번개 3발 → 2발 또는
  - caff axe cd 0.7 → 0.9
- **근거**: monhun 풀투자 + 황기훈 시너지 — 15분 보스 1~2초 컷이 일상

### 🚨 P0-2: 최종원 후반 폭주 시너지 과다
- **현황**: 시간 스케일링 `dm = 1 + (t/120)×0.7`
  - 10분 시점: dm = 4.5
  - + recipe Lv5 (×1.2): 5.4
  - + knife2 Lv5 (×1.15): 6.21
  - + wdmg Lv5 (×1.25): 7.76
  - + 폭주 시 공속 ×2 → **effective DPS 15배 이상**
- **권장**:
  - 시간 스케일링 0.7 → **0.4** (10분 시점 dm 4.5 → 3.5)
  - 또는 후반에 자동 다른 캐릭(0.5 max)으로 cap
- **근거**: 패시브 풀스택 시너지 누적이 폭발적

## ✅ 잘 디자인된 부분

### 회복 시스템 (pill + combat lock)
- pill 만렙 7.5 HP/s + 2초 combat lock
- 후반 좀비 30 dmg × 1.25hit/s = 37.5 DPS > regen 7.5
- 무한 버티기 불가능 + 비전투 시간 보상 — 잘 작동

### 크리 시스템 (너프 후)
- 최초 베이스 20% + bark Lv5 (60%) = 80% 만렙
- 실효 데미지 배수 ×1.8 — recipe/knife2 같은 다른 빌드와 비슷한 수준

### 룰렛 확률 (재분배 후)
- 꽝 15% (4번에 1번) → 짜증 적당
- 잭팟 6% (17번에 1번) → 운 좋은 발견 느낌
- 일반 5명 79% → 안정적 다양성

### 컴패니언 (최초 비글)
- 단일 타깃 추가 DPS 24/0.75 = 32 (Lv1)
- Lv2+ 할퀴기 / Lv3+ 짖기로 다중 타깃 보조
- 자동 운영으로 안정적 — OP 아닌 신뢰성 빌드

## 📝 마치며

지금 BAL 중앙화 덕분에 위 권장사항들은 **한 줄씩 BAL 객체 값만 바꾸면 적용**돼요. 예:

```js
// P0-1 권장
BAL.wpn.caff.lightningDmgMul = 1.2;  // 1.8 → 1.2
// 또는
BAL.wpn.caff.lightningCount = 2;  // 3 → 2

// P0-2 권장
BAL.char.jw.lateScaleRate = 0.4;  // 0.7 → 0.4

// P1-1 권장
BAL.wpn.light.dmgMul = 1.0;  // 0.78 → 1.0
```

위 권장 조치 다 적용하고 한 번 더 플레이 테스트 하고, 그 결과 보고 추가 조정해도 좋고요.

엑셀 파일 (`balance_analysis.xlsx`) 열어보면:

- 시트 3: 적 능력치 시간대별 곡선
- 시트 4: 무기별 TTK (적 종류 vs)
